Welcome to the Maze Project!

An innovative maze game built using SDL2 and Raycasting, designed to challenge your navigation skills!

Play Now!

Key Features

Camera Rotation

Rotate the camera using the left and right arrow keys, as well as mouse movements, to explore the maze in a first-person perspective.

Movement Controls

Move within the maze using the W, A, S, and D keys to navigate through complex paths and avoid obstacles and some scary enemies.

Dynamic Weapon System

Change weapons by pressing the number keys (1-4), each corresponding to a different weapon, adding strategy to your gameplay.

About This Project

This project was inspired by my love for puzzle games and my desire to create an engaging 3D maze experience. It has been a fantastic journey of learning SDL2 and implementing raycasting techniques.

This is a Portfolio Project for Holberton School, showcasing my skills and creativity. Check it out on my GitHub repository.
